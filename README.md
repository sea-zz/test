# haiyan
一些练习、学习、项目demo、闲鱼等

## blog-egg-api
基于egg开发，连接本地数据库，请求第三方接口，上传功能等
```
安装
cnpm i
启动
cnpm run dev
```

## blog-umi
基于umi、antd开发
```
安装
pnpm i
启动
pnpm run dev
```

### pnpm命令
```
添加
pnpm add 包名  // 默认保存到 dependencies 配置项下
-D // 保存到 devDependencies 配置项下
删除
pnpm remove 包名
更多参考官网：https://www.pnpm.cn/cli/remove
```

## 发布
```
// api服务启动和停止
cnpm run start 
cnpm run stop

// 网页发布 
pnpm run build // 打包后直接放到目录访问即可
```
